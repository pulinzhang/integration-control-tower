# Integration Control Tower - UI/UX Design Specification

## 1. Design System Overview

### 1.1 Design Philosophy

- **Style**: Enterprise SaaS control tower (like Datadog, New Relic, AWS CloudWatch)
- **Theme**: Dark mode with high contrast for data readability
- **Persona**: Professional, trustworthy, sophisticated

### 1.2 Color Palette

| Role                     | Color         | Hex Code  | Usage                            |
| ------------------------ | ------------- | --------- | -------------------------------- |
| **Background Primary**   | Obsidian      | `#0D0D12` | Main background                  |

| **Background Tertiary**  | Slate         | `#1E1E28` | Hover states, elevated surfaces  |
| **Border**               | Graphite      | `#2A2A36` | Dividers, card borders           |
| **Text Primary**         | Snow          | `#F4F4F6` | Headings, primary content        |

| **Text Muted**           | Pewter        | `#6B7280` | Placeholders, disabled           |

- **Base unit**: 4px
